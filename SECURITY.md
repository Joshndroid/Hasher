# Security policy

Please report suspected vulnerabilities through GitHub's private vulnerability reporting
for this repository. Do not open a public issue containing exploit details, sensitive
forensic material, signing credentials, or unredacted evidence metadata.

Release binaries should be trusted only when their SHA-256 value appears in the release's
`SHA256SUMS.txt` and their GitHub provenance attestation verifies for this repository.
See [the GitHub release security setup](docs/github-release-security.md) for the complete
release controls and maintainer configuration.
